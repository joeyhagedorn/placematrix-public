Place Matrix

[archive]

You can toggle one LED immediately and watch it reflected in the live video stream above. Your next chance is based on the number of participants. There is a slight delay in the live stream which adds to the challenge.

Select an LED to toggle.

This project was inspired by Reddit's April 1 art project Place (about). I decided to make a more tangible version; this one uses The Matrix and a Particle Photon to toggle real life LEDs in my kitchen.

While this is no longer live, you can read about the project here, and click around the simulated page.

Created by Joey Hagedorn. Special thanks to Charlie Deets, Reddit, and Boldport Club.
